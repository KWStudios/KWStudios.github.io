Tutorials

who we are

It is a little complicated to play on our modded survival server, but the work pays off. Nowhere else can a Modpack with the unique and amazing Tech-guns mod be found. So here's a small guide on how to join us.

1. Download and install the ATLauncher

2. Open it, log in with your account and go to the "Packs" tab

3. Search for "Bevo's Techpack" and klick "New Instance". A menu pops up with "Instance name", "Version", and "enable user lock".
Name the instance whatever you want, ignore user lock and chosse the Version BTP-Reborn-BetaX-Hotfix. Some warnings may pop up, ignore them

4. On the bottom of the installation menu, choose "use share code" and enter the code "oRm7bI0D" (without the ""). Click install after you entered the code

5. Go to the tab "Instances" after the installation finished. You will see the installed instance "Bevo's Techpack". Click on the button "Open Folder".

6. Move into the folder "mods". Delete "Galacticraft-Core", "Galacticraft-Planets", "Chisel", and "MicDoodle-Core".

7. Download the following mods for 1.7.10:
Techguns
Agricraft
Big Reactors
Brandon's Core
Chisel
DraconicEvolution
DragonAPIDragonAPI
ElectricraftElectricraft
FTBLibFTBLib
FTBUtilitiesFTBUtilities
Galacticraft(download all three .463 versions of the mod components)
Minechem
Reactorcraft
Rotarycraft
make sure to download the latest 1.7.10 versions of the mods.

8. Unzip the files into the mods folder of the modpack. If you have done everything right, you can start the game in the instance tab of your ATLauncher and join our server.
